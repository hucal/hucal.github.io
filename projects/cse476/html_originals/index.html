Trade Visualizations

These are two very different visualizations of the Correlates of War Trade data. I created a conventional geographic visualization and a Hive Plot based alternative. The third one is a joint visualization using both techniques.

Source: Bilateral Trade v3.0 from the Correlates of War. See the trade/data directory for the files. Note, I only uploaded data for every three years starting form 1870 and ending in 2008. Also helpful is this Makefile. It should help you download all the sources. One of these must be done manually, the cshapes data was used for historical maps. Need this python script.
